# 🤖 Jarvis Virtual Assistant

A Python-based voice assistant that performs tasks like opening apps, searching the web, and automation.

## 🚀 Features
- Voice commands
- Task automation
- Web search

## 🛠 Tech Stack
- Python
- Speech Recognition
- pyttsx3

## ⚠️ Note
This project is designed for local system use and is not deployed.
